# Commands API

## Fetch Commands
- **Endpoint**: `/api/admins/commands`
- **Method**: `GET`

## Create Command
- **Endpoint**: `/api/admins/commands`
- **Method**: `POST`

## Update Command
- **Endpoint**: `/api/admins/commands/:commandId`
- **Method**: `PUT`

## Delete Command
- **Endpoint**: `/api/admins/commands/:commandId`
- **Method**: `DELETE`
