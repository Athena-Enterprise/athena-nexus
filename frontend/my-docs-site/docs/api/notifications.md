# Notifications API

## Fetch Notifications
- **Endpoint**: `/api/admins/notifications`
- **Method**: `GET`

## Create Notification
- **Endpoint**: `/api/admins/notifications`
- **Method**: `POST`

## Update Notification
- **Endpoint**: `/api/admins/notifications/:notificationId`
- **Method**: `PUT`

## Delete Notification
- **Endpoint**: `/api/admins/notifications/:notificationId`
- **Method**: `DELETE`
